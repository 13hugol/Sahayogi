# US-30 / F3-3.6 - Location Filter Implementation Report

## User Story

As a registered or guest user, I want to filter listings by my location and a distance radius so that I can find skills offered near me.

## Completion Status

Completed on the `Prashna` branch. The system allows searching listings by city names or custom latitude, longitude coordinates, with radius limits (10km, 50km, 100km). Calculated distances are cached on the listings and dynamically displayed on the listings cards.

## Implementation Summary

- **Utility Module**: Created `app/utils/distance.py` containing coordinate mappings for Nepal's cities and a `haversine_distance` formula implementation.
- **Frontend Controller**: Updated `FrontendController.marketplace` and `FrontendController.api_search` to parse search locations, fall back to owner profile locations if listing location is empty, calculate distances using the Haversine formula, and filter results by radius.
- **Template Updates**:
  - Added a Location input field and a "📍 Detect Location" button in `app/templates/listings/index.html` using the HTML5 Geolocation API.
  - Added active filters display with clear buttons in `app/templates/listings/index.html`.
  - Updated `app/templates/partials/listing_cards.html` to show the distance dynamically (e.g. `(4.8 km away)`) when calculated.

## Acceptance Criteria Evidence

| Acceptance Criterion | Implementation Evidence |
| --- | --- |
| Search by city or coordinates | Works with cities listed in the Nepal cities coordinates mapping (e.g., Kathmandu, Pokhara, Lalitpur) or literal coordinate input (e.g., `27.7172, 85.3240`). |
| Radius Filtering | Filters results to listings within 10km, 50km, or 100km of the search origin. If radius is omitted ("Any"), distance is still computed and displayed without radius filtering. |
| User Profile Fallback | If a listing has an empty `location_text`, the system automatically falls back to using the owner's profile location. |
| Fuzzy Text Fallback | If the location input cannot be mapped to coordinates, the system falls back to literal substring text matching on the location field. |
| Display Distance | Listing cards display the calculated distance formatted to one decimal place next to the location text. |
| Geolocation Auto-Detection | Clicking "📍 Detect Location" requests device coordinates via the Geolocation API and auto-fills the input field. |
